# ITRI-project <br/> LTE-A/Wi-Fi  雙模接取小型基站自動優化技術       

## Objective
基於工研院的LTE Small cell System 設計一套自動優化演算法，藉由調整多個 Smart Antenna 之水平及垂直角度和功率強度，來同時達到Coverage 和 Capacity   的最佳化
